# Parijs site v0.06

## Start lokaal

```powershell
npm install
$env:ADMIN_USERNAME="admin"
$env:ADMIN_PASSWORD="kies-hier-een-sterk-wachtwoord"
$env:SESSION_SECRET="lange-unieke-sessie-sleutel"
npm start
```

Open daarna `http://localhost:3000`.

## Render

Build command: `npm install`
Start command: `npm start`
